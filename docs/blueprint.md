# **App Name**: HealthStreak

## Core Features:

- User Authentication: Allow users to log in using their mobile number and email ID.
- Profile Management: Enable users to upload a profile photo, set wake-up time (4-7 AM with alarm).
- Daily Task Notification: Remind users to do tasks (jogging, yoga, etc.) as per their preferences. The LLM tool determines whether or not to include exercise reminders based on the user's set preferences.
- Photo Verification: Require users to take a photo of their morning environment (like nature) as proof of waking up.
- Customized Diet Plan: Provide personalized diet plans based on user details (weight, height).
- Streak Tracking and Redemption: Track daily task streaks, allow redemption for Indian Rupees (INR).
- Progress Levels: Assign levels (Beginner, Intermediate, etc.) based on user participation.

## Style Guidelines:

- Primary color: Forest green (#337053) to evoke health, growth, and nature.
- Background color: Light beige (#F5F5DC) to provide a neutral, earthy feel.
- Accent color: Burnt orange (#CC5500) to highlight calls to action and important information.
- Body font: 'PT Sans', a humanist sans-serif
- Headline font: 'Playfair', a modern serif. Use 'PT Sans' for body text.
- Use line icons for tasks and health metrics, ensuring they are clean and easily recognizable.
- Implement a clean and intuitive layout with clear sections for tasks, profile, and rewards.
- Subtle transitions and animations to provide feedback on user actions, such as task completion.
